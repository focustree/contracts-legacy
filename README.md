## Setup

1. [Install protostar](https://docs.swmansion.com/protostar/docs/tutorials/installation).
2. Build the project:

```sh
protostar build
```

3. Run the tests:

```sh
protostar test
```

## Commands

### Compile cairo contracts

```sh
protostar build
```

### Run tests

```sh
protostar test
```

### Format source code

```sh
protostar format
```

### Install deps

```sh
protostar install OpenZeppelin/cairo-contracts@v0.4.0
```
